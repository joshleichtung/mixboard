Architecture review for {{system_name}} from Senior Director perspective:

**Review Framework:**

1. **System Design Quality**
   - Scalability: Can it handle 10x growth?
   - Maintainability: Can new engineers understand it in < 1 week?
   - Testability: Are critical paths covered?
   - Observability: Can we debug production issues?

2. **Technical Debt Assessment**
   - Current debt level (Low/Medium/High)
   - Impact on velocity
   - Recommended refactoring priorities

3. **Health Tech Specific**
   - HIPAA compliance architecture
   - PHI data flow and encryption
   - Audit logging coverage
   - Disaster recovery planning

4. **Team & Process**
   - Documentation quality
   - Onboarding friction points
   - Deployment confidence level
   - On-call burden assessment

5. **Strategic Alignment**
   - Supports business roadmap?
   - Technology choices future-proof?
   - Build vs buy decisions sound?

**Output Format:**
- Architecture diagram (if needed)
- Risk matrix
- Specific recommendations with priority
- Resource requirements for improvements
